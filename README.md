# AI Web Agency

AI-assisted workflow for planning, building, and improving modern websites.
This repository is designed to manage web projects using AI tools while the developer acts primarily as a supervisor.

## Overview

This project demonstrates a development workflow where artificial intelligence handles most of the coding, debugging, and refactoring tasks, while the developer focuses on planning, reviewing, and guiding the project.

AI tools used in this workflow include Claude for architecture and planning, OpenAI Codex for code editing and debugging, and Google Gemini for generating UI components.

Projects are managed and version-controlled using GitHub.

## Goals

* Reduce manual coding
* Accelerate website development
* Maintain clean and organized codebases
* Use AI tools efficiently in real-world web projects

## AI Development Workflow

1. Planning
   Define the website structure and features.

2. Code Generation
   Generate HTML, CSS, and components using AI tools.

3. Integration
   Merge generated code into the project structure.

4. Debugging
   Use AI to identify and fix bugs or layout issues.

5. Refactoring
   Improve code quality and maintainability.

6. Version Control
   Commit and track all changes using GitHub.

## Example Tasks
